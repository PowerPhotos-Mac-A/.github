## Overview

PowerPhotos Advanced is a photo library management tool from Fat Cat Software that provides capabilities that Apple Photos lacks — browsing multiple libraries simultaneously, merging libraries, finding and removing duplicate photos, and migrating content between libraries with metadata preservation. It addresses the limitations that Photos imposes on library management, particularly the inability to work with more than one library at a time and the absence of duplicate detection. The application works alongside Photos rather than replacing it, enhancing the organizational capabilities that Apple's application does not provide.

Multiple library browsing displays the contents of any Photos library without switching the system library, enabling comparison, organization, and content review across multiple libraries from a single interface. The browser shows albums, smart albums, and media types from any library regardless of which library Photos is currently using. This capability is essential for users who maintain separate libraries for different purposes — personal, professional, archive — and need to view and manage them without constant library switching.

Library merging combines two or more Photos libraries into a single consolidated library with intelligent duplicate handling that identifies and resolves matching photos rather than creating duplicates in the merged result. The merge preserves albums, smart albums, keywords, favorites, and other metadata from both source libraries. Conflict resolution options handle cases where the same photo exists in both libraries with different edits or metadata.

Duplicate detection scans Photos libraries for matching images using visual comparison that identifies duplicates even when file names, dates, or resolutions differ. The comparison algorithm handles photos that have been resized, cropped, or slightly edited, catching duplicates that filename-based detection would miss. The duplicate review interface shows matched pairs side by side with metadata comparison, enabling informed decisions about which copy to keep.

For anyone managing large or multiple Apple Photos libraries who needs the organizational tools that Photos itself does not provide — especially duplicate removal and library merging — PowerPhotos Advanced is the library management tool that fills the gaps in Apple's photo management ecosystem. Its non-destructive approach ensures that library operations never risk the original photo data.

## Performance

PowerPhotos balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
